<h1 align="center">Sudoku Solver</h1>
This is a JavaScript-based Sudoku Solver application that allows users to input a Sudoku puzzle and get the solved version.

## Features
* Solves Sudoku puzzles using backtracking algorithm

* User-friendly interface for inputting puzzle data


## Usage
* Enter the numbers for your Sudoku puzzle in the provided grid. Leave blank cells empty.

* Click the "Solve Puzzle" button.
